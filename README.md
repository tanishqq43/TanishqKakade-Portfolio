# Tanishq Kakade — Portfolio

My personal portfolio website — a single page that shows who I am, what I can build, and how to reach me.

🔗 **Live site:** https://tanishq-kakade-portfolio.vercel.app/

## What's inside

- **About** — quick intro about me
- **Skills** — Java, Hibernate, JDBC, React, SQL/PostgreSQL, Git & tools
- **Experience** — internship and work history
- **Certifications** — courses and certificates I've completed
- **Projects** — including RK Motors, an AI-powered vehicle marketplace
- **Resume** — downloadable resume
- **Contact** — email, GitHub, and LinkedIn

## Built with

Just the basics — no frameworks, no build step:

- `HTML5`
- `CSS3`
- `JavaScript`

## Running it locally

No installation needed. Just clone and open the file:

```bash
git clone https://github.com/tanishqq43/TanishqKakade-Portfolio.git
cd TanishqKakade-Portfolio
```

Then open `index.html` in your browser — that's it.

## Folder structure

```
├── index.html   # all the page content
├── style.css    # all the styling
```
